Research

Using machine learning to detect indicators of water on Mars

With the Jet Propulsion Laboratory's Machine Learning and Instrument Autonomy (MLIA) Group, I studied recurring slope lineae (RSL), a Mars landform that may indicate liquid water. I worked with a machine learning pipeline that automatically detects RSL in high-resolution images from Mars orbiters. My main contribution is to cluster Mars terrains into those "likely" and "unlikely" to form RSL, to improve the performance of the pipeline.

Network monitoring system for delay-tolerant networks

As an intern at the Jet Propulsion Laboratory in 2016-17, I built a network monitoring system for ION, an experimental networking suite for communications in deep space. The system is currently used to monitor a test ION network that spans multiple NASA centers. I wrote a C extension to ION, and then designed the system using Elasticsearch, Logstash, and Kibana.

Projects

TrekStar Route Planner

For a networking capstone project, our 5-person team built an online route planner. We noticed that Google Maps can only make shortest paths from origin to destination. To make more exciting routes, our app took preferences for types of roads (steepness, amount of roadside greenery), preferences for types of points of interest (restaurants, coffee shops, etc.), and a maximum distance. This involved solving interesting algorithms such as the orienteering algorithm.

Report

Replication in relational database

As a database capstone project, I added replication to Caltech's teaching relational database NanoDB. I noticed that NanoDB ran only on one machine, and replication would increase its reliability. I studied replication approaches in prominent relational databases such as Postgres and MySQL. Then I implemented primary-standby replication from Java sockets up.

Underwater Robot - Dory

With the Caltech Robotics Team in 2015-16, we built an autonomous underwater vehicle for the AUVSI RoboSub competition and won 1st place among 46 teams. We have an extensive C++ codebase that provides the autonomous "smarts" for the robot. My contributions were:

  • Implemented autonomously finding and removing the cover of an underwater bin, a competition task. Used C++ and OpenCV for computer vision.
  • Wrote the interface to send commands to the robot's thrusters, a vital piece of middleware, in C++.

Team website: http://crt.caltech.edu

Unix-like operating system

With a 3-person team, I developed a Unix-like teaching operating system for Operating Systems class in under ten weeks. We implemented kernel threads, virtual processes, paged virtual memory, and a filesystem using C and x86 assembly.
