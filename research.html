Machine Learning and Instrument Autonomy (MLIA) Group

NASA Jet Propulsion Laboratory

Pasadena, California

I am applying machine learning to Mars terrain data to do automatic terrain classification. The specific goal is to learn more about recurring slope lineae (RSL)--a little-understood Mars landform that may be caused by liquid water! Since this summer, I have been using Python machine learning libraries (Scikit-Learn, Numpy) to do many terrain classification experiments (mainly k-means clustering) on Mars terrain maps, which can have millions of pixels. Some of the initial results have been promising, and I am continuing to work on the project during the school year.
